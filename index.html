McCabe-Thiele 선도
VLE 곡선, 조작선, 단계선을 실시간으로 그립니다. 각 요소를 켜고 끄며 이론을 단계별로 이해할 수 있습니다.
그래프 토글
다중 증류탑 설계
탑을 직렬로 추가하고, 이전 탑의 탑저가 다음 탑의 공급으로 자동 연결됩니다. 복잡한 공정 흐름을 직관적으로 구성합니다.
공정 흐름 빌더
Mixture / Pure Flow 시각화
각 스트림이 혼합물인지 순수 물질인지를 색상과 흐름 애니메이션으로 구분합니다. 물질 수지를 눈으로 추적할 수 있습니다.
실시간 흐름 표시
직접 값 입력 계산
파라미터를 슬라이더가 아닌 숫자로 직접 입력합니다. 데이터시트 값을 그대로 넣으면 이론 단수, 회수율이 즉시 계산됩니다.
입력형 파라미터
공정 진단 피드백
Rmin 미달, 단수 과다, 조성 충돌 등을 실시간으로 감지하고 학습 가이드와 함께 경고합니다.
실시간 진단

환류비 R (Reflux Ratio)
탑 상부에서 다시 탑으로 돌려보내는 액체(환류액)와 제품으로 뽑아내는 양의 비율.

R = L / D

R 크면 → 분리 성능↑, 에너지↑
R < Rmin → 분리 불가능
공급 조성 xF
공급 원료에서 분리 대상 성분(경성분, 예: 에탄올)의 몰분율.

xF=0.35 → 공급물 35%가 에탄올
탑상 조성 xD (Distillate)
탑 상단 제품의 목표 순도(몰분율).

xD=0.85 → 제품의 85%가 에탄올

값이 클수록 더 순수하지만 더 많은 단수·환류비 필요.
탑저 조성 xB (Bottoms)
탑 하단 잔류물의 경성분 몰분율.

xB=0.05 → 잔류물 5%만 에탄올, 95%는 물
공급 상태 q (Feed Quality)
공급 원료의 열적 상태.

q=0 : 포화 증기
q=1 : 포화 액체 (버블 포인트)
q>1 : 과냉 액체
0<q<1 : 부분 기화
